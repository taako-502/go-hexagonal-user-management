# go-sample-api

以下のコマンドで実行する。

```bash
$ export PORT={ポート番号}
$ go run main.go
```

## 使い方

### 環境

http://localhost:6000/

### API 仕様

例）

```
http://localhost:6000/get-psi-performance?url=https://hepere.com
```
